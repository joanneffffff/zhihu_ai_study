🔍 Embedding Query和Chunk矩阵乘法详解

1. 示例代码

sentences_1 = ["What is BGE M3?", "Defination of BM25"] # Query sentences_2 = ["BGE M3 is an embedding model...", "BM25 is a bag-of-words..."] # Chunk embeddings_1 = model.encode(sentences_1)['dense_vecs'] # Query embeddings embeddings_2 = model.encode(sentences_2)['dense_vecs'] # Chunk embeddings similarity = embeddings_1 @ embeddings_2.T # 矩阵乘法

2. 矩阵维度说明

BGE-M3模型的dense embedding维度： 1024维 (d = 1024)
1 Query Embeddings (embeddings_1)
形状: (2, 1024)
  • 2个query句子
  • 每个句子编码为1024维向量
2 Chunk Embeddings (embeddings_2)
形状: (2, 1024)
  • 2个chunk文档
  • 每个chunk编码为1024维向量
3 转置操作 (embeddings_2.T)
形状: (1024, 2)
  • 将chunk embeddings转置
  • 从 (2, 1024) 变为 (1024, 2)

4. 矩阵乘法计算过程

矩阵乘法公式：
similarity[i, j] = Σ(k=0 to 1023) embeddings_1[i, k] × embeddings_2[j, k]
具体计算示例：
位置 计算公式 含义
similarity[0, 0] q₁ · c₁ = Σ q₁[k] × c₁[k] Query1与Chunk1的余弦相似度
similarity[0, 1] q₁ · c₂ = Σ q₁[k] × c₂[k] Query1与Chunk2的余弦相似度
similarity[1, 0] q₂ · c₁ = Σ q₂[k] × c₁[k] Query2与Chunk1的余弦相似度
similarity[1, 1] q₂ · c₂ = Σ q₂[k] × c₂[k] Query2与Chunk2的余弦相似度

8. 数学原理

向量点积（内积）的几何意义：
  • 两个向量的点积 = ||q|| × ||c|| × cos(θ)
  • 其中θ是两向量之间的夹角
  • 点积越大，夹角越小，相似度越高
为什么使用转置？
  • embeddings_1形状: (2, 1024) - 2个query，每个1024维
  • embeddings_2形状: (2, 1024) - 2个chunk，每个1024维
  • 转置后embeddings_2.T形状: (1024, 2)
  • 矩阵乘法规则: (2, 1024) @ (1024, 2) = (2, 2)
  • 结果矩阵的每个元素[i,j] = query[i]与chunk[j]的点积

9. SVG可视化：矩阵乘法过程

💡 关键理解：
矩阵乘法 embeddings_1 @ embeddings_2.T 实际上是在计算所有query和chunk之间的成对相似度。 结果矩阵的每一行对应一个query，每一列对应一个chunk，矩阵元素的值就是它们之间的相似度分数。
