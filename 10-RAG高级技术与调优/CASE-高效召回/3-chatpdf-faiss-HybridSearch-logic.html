核心代码结构

class HybridRetriever: def __init__(self, chunks, vectorstore, alpha=0.5) def search(self, query, k=4) 主要功能函数: - extract_text_with_page_numbers(): PDF文本提取 - tokenize_chinese(): 中文分词 - process_text_with_splitter(): 文本分块和向量化 - generate_multi_queries(): 生成查询变体 - hybrid_multi_query_search(): 混合多查询检索
